The Skin Genie ritual

Four steps. Clean, gentle, consistent.

Every Skin Genie tool is built around the same idea: a simple, well-cared-for routine you can actually keep up with.

01

Cleanse

Start with freshly cleansed, dry skin and clean hands. Reset the surface before anything else.

02

Prepare

Sanitize the tool before use. Choose the right setting or roller for the ritual you're doing today.

03

Use gently

Light pressure, slow movement. Avoid irritated, broken or uncomfortable skin — always use as directed.

04

Clean & store

Sanitize again, dry fully, return to the included holder. A clean tool is the most important detail.

Good to know

Common questions, answered.

Can I use Skin Genie tools every day?

It depends on the tool. The Electric Gua Sha is designed to fit easily into a frequent routine. Derma rollers should be used only as directed, with appropriate recovery time between sessions.

Can I use a derma roller on irritated skin?

No. Avoid broken, irritated, inflamed or uncomfortable skin. Always start on clean, dry skin and stop use if anything feels off.

Can I share my derma roller?

No. Derma rollers are personal-use tools and should never be shared. Each roller is intended for one person, with proper cleaning and storage.

Should I start with 0.5mm or 1.5mm?

Start with 0.5mm if you're new to derma rolling. The 1.5mm is intended for experienced users who already know their cadence and follow the instructions carefully.
